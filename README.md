# Lab 04 Java Constructor Tasks

## Task 1: LibraryBook Class
- Default values if no data provided:
  - `bookTitle = "Unknown"`
  - `price = 0`
- If title provided only, `price = 500`
- If title and price provided, assign both values
- Create objects using all constructors

- Use a copy constructor to create a safe duplicate
- Changes to the duplicate must **not** affect the original
